# JobHub Pro

JobHub Pro is a production-style job portal application that connects job seekers and recruiters.

## Features

- JWT Authentication
- Role Based Access Control
- Company Management
- Job Posting
- Job Applications
- Resume Upload
- Dashboard Analytics

## Tech Stack

Backend:
- Java 21
- Spring Boot
- Spring Security
- Hibernate
- MySQL

Frontend:
- React
- React Router
- Axios

Version:
v1.0
